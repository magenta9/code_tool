## Usage

- **In Progress**: 当前正在开发的任务
- **Completed**: 已完成的任务
- **Backlog**: 待处理的任务，按优先级排序

### Update Format

When updating status, use this format:

```
### YYYY-MM-DD

**In Progress:**
- [Task name] - 状态更新

**Completed:**
- [Task name] - 完成说明
```
